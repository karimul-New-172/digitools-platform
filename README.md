# 🚀 DigiTools Platform

**DigiTools** is a modern premium digital tools platform where creators and professionals can browse essential productivity and creative tools. Users can explore various products, add them to their cart, and proceed to checkout seamlessly.

---

## ✨ Core Features

- **Premium Tools Catalog:** A curated collection of digital products organized in a clean card layout.
- **Interactive Cart System:** Real-time functionality to add/remove items and calculate the total price.
- **Dynamic State Management:** Built with React `useState` for a smooth and reactive user experience.
- **Responsive Navigation:** Mobile and desktop-friendly navbar featuring a live cart item counter.
- **Modern UI:** Designed with Tailwind CSS for a professional, high-quality look and feel.

---

## 🛠 Tech Stack

This project was built using:

- **Frontend Library:** [React.js](https://reactjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Icons:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
